Deploy de Modelos com Flask

Descubra como transformar um modelo de Machine Learning em uma aplicação web funcional e acessível, utilizando o framework Flask.

1. O que é o Flask?

Flask é um microframework em Python, leve e flexível, ideal para criar APIs e interfaces web para aplicações de machine learning.

2. Preparação do modelo

Treine seu modelo usando Scikit-learn, TensorFlow, etc. Em seguida, salve-o com joblib ou pickle para reutilização.

3. Estrutura do projeto

Organize seu projeto com uma pasta principal contendo o arquivo app.py, a pasta templates e o modelo salvo.

4. Código básico do Flask

from flask import Flask, request, jsonify\nimport joblib\nmodel = joblib.load('modelo.pkl')\napp = Flask(__name__)\n@app.route('/predict', methods=['POST'])\ndef predict():\n    data = request.get_json()\n    prediction = model.predict([data['features']])\n    return jsonify({'prediction': prediction.tolist()})

5. Testando com Postman

Use ferramentas como Postman ou curl para enviar dados via JSON e testar sua rota /predict.

6. Front-end com HTML

Crie um formulário básico em HTML e use o Flask para renderizá-lo com render_template.

7. Deploy local

Rode sua aplicação localmente com flask run e acesse via http://localhost:5000.

8. Deploy na nuvem

Use plataformas como Heroku, Vercel ou Render para colocar sua API Flask online com poucos comandos.

9. Requisitos

Inclua um requirements.txt com todas as bibliotecas necessárias para o funcionamento da aplicação.

10. Considerações finais

Com Flask, é possível criar e expor modelos inteligentes de forma elegante e funcional, pronto para uso em sistemas reais.
