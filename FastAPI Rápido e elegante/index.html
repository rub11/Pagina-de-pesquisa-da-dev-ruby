FastAPI: Rápido e Elegante

Aprenda a criar APIs RESTful rápidas e elegantes com o FastAPI, um framework Python moderno e altamente performático. Abaixo, exploramos os fundamentos e exemplos de código para começar.

O que é FastAPI?

FastAPI é um framework moderno para a construção de APIs com Python, projetado para ser rápido, fácil de usar e com excelente suporte para a criação de APIs RESTful. Ele aproveita recursos como tipagem estática, validação automática e execução assíncrona.

Vantagens do FastAPI

Algumas das principais vantagens do FastAPI incluem:

  • Desempenho elevado (comparável ao Node.js e Go).
  • Validação automática de dados e documentação interativa com Swagger.
  • Suporte a assíncrono, o que o torna ideal para aplicativos de alta concorrência.
  • Tipagem estática do Python para maior confiabilidade no código.

Instalação do FastAPI

Para começar a usar o FastAPI, basta instalar o pacote com o pip. Também precisaremos de um servidor ASGI, como o Uvicorn, para executar nossa aplicação.

pip install fastapi uvicorn

Criando a Primeira Rota

Agora, vamos criar uma aplicação simples com FastAPI que responde a uma requisição GET.

from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def read_root():
    return {"message": "Olá, FastAPI!"}

Executando a API

Para executar nossa API, basta usar o Uvicorn para rodar a aplicação FastAPI.

uvicorn main:app --reload

O servidor estará disponível em http://127.0.0.1:8000.

Validando Dados com FastAPI

FastAPI permite validar dados de entrada automaticamente usando Pydantic. Isso nos ajuda a garantir que os dados recebidos pelas rotas sejam válidos e no formato correto.

from pydantic import BaseModel

class Item(BaseModel):
    name: str
    price: float

@app.post("/items/")
def create_item(item: Item):
    return {"name": item.name, "price": item.price}

Documentação Interativa

Uma das características mais notáveis do FastAPI é a documentação interativa automática, que pode ser acessada em /docs. Com ela, você pode testar sua API diretamente na interface.

Assincronismo no FastAPI

O FastAPI oferece suporte total ao desenvolvimento de APIs assíncronas, o que melhora a escalabilidade e o desempenho, especialmente em aplicativos com alta carga de solicitações.

import asyncio

@app.get("/async")
async def async_endpoint():
    await asyncio.sleep(1)
    return {"message": "Resposta assíncrona!"}

Conclusão

O FastAPI é uma ferramenta poderosa para desenvolvedores que buscam construir APIs rápidas, escaláveis e com código limpo. Com sua facilidade de uso, tipagem estática e suporte a assíncrono, ele se destaca como uma das melhores opções para criar APIs com Python.
