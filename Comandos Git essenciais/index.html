Comandos Git Essenciais

Conheça os comandos mais usados no dia a dia do desenvolvedor

1. Inicializar um repositório

Cria um novo repositório Git na pasta atual.

git init

2. Clonar um repositório

Cria uma cópia local de um repositório remoto.

git clone https://github.com/usuario/repositorio.git

3. Verificar o status

Exibe arquivos modificados, adicionados e não versionados.

git status

4. Adicionar arquivos ao stage

Prepara arquivos para o commit.

git add . git add nome-do-arquivo.txt

5. Criar um commit

Salva as alterações com uma mensagem descritiva.

git commit -m "mensagem explicativa do que foi feito"

6. Ver histórico de commits

Lista todos os commits realizados no projeto.

git log

7. Enviar alterações para o repositório remoto

Atualiza o repositório online com seus commits locais.

git push origin main

8. Atualizar repositório local

Baixa e integra as últimas mudanças do repositório remoto.

git pull

9. Criar e mudar de branch

Cria uma nova ramificação e alterna para ela.

git checkout -b nova-feature

10. Mesclar branches

Une o conteúdo de uma branch à branch atual.

git merge nome-da-branch
