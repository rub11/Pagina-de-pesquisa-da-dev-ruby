Quiz sobre Cloud Security

O que é Cloud Security?

  • Conjunto de políticas e controles para proteger dados, aplicações e serviços na nuvem.
  • Tecnologia usada para otimizar a performance de servidores na nuvem.
  • Método de backup para dados em servidores físicos.

Qual é o objetivo principal da criptografia em Cloud Security?

  • Proteger dados confidenciais de acessos não autorizados durante a transmissão ou armazenamento.
  • Acelerar a velocidade de transferência de dados.
  • Reduzir o uso de espaço de armazenamento na nuvem.

Qual é a principal vantagem do uso de multi-fatores na autenticação em ambientes de nuvem?

  • Aumenta a segurança ao exigir mais de uma verificação para acesso ao sistema.
  • Facilita o acesso ao sistema sem necessidade de senhas.
  • Reduz o tempo necessário para autenticação dos usuários.

O que é uma ameaça persistente avançada (APT) em Cloud Security?

  • Ataques de longa duração e altamente organizados, geralmente com o objetivo de roubar dados ou acessar informações sensíveis.
  • Ataques que ocorrem exclusivamente em servidores locais e não afetam a nuvem.
  • Ataques realizados por usuários não autorizados que buscam acessar informações de maneira simples.

O que é o conceito de "segurança por design" em Cloud Security?

  • Integrar a segurança desde a concepção e desenvolvimento de sistemas na nuvem.
  • Aplicar segurança apenas após a implementação do sistema na nuvem.
  • Utilizar apenas firewalls para proteger as redes na nuvem.

O que são controles de acesso baseados em identidade (IAM) em ambientes de nuvem?

  • Sistema que define quem pode acessar quais recursos da nuvem, com base em permissões atribuídas a usuários específicos.
  • Ferramentas que protegem dados com criptografia em tempo real.
  • Serviços de backup para proteger a infraestrutura na nuvem.

O que é o modelo de "defesa em profundidade" em Cloud Security?

  • Uso de várias camadas de defesa para proteger dados e sistemas contra ameaças.
  • Uma única camada de segurança suficiente para a proteção de dados.
  • Um modelo que não requer a utilização de criptografia em ambientes de nuvem.

O que é a auditoria de segurança em ambientes de nuvem?

  • Análise contínua dos acessos, atividades e configurações em ambientes de nuvem para identificar possíveis vulnerabilidades ou falhas de segurança.
  • Ação de bloquear todos os acessos externos ao sistema.
  • Ferramenta de análise de performance da rede de dados na nuvem.

Qual é a função da segmentação de rede em Cloud Security?

  • Dividir a rede em segmentos menores para melhorar o controle de tráfego e reduzir riscos de ataque.
  • Aumentar a largura de banda disponível para os usuários.
  • Criar uma rede redundante para garantir continuidade de serviço.

O que significa a sigla "DLP" em Cloud Security?

  • Data Loss Prevention (Prevenção de Perda de Dados), um conjunto de políticas e ferramentas para evitar o vazamento de informações confidenciais.
  • Dynamic Load Prevention (Prevenção de Carga Dinâmica), um sistema para balanceamento de carga em servidores.
  • Digital Login Protection (Proteção de Login Digital), ferramenta para autenticação em dois fatores.
