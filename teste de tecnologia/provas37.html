Quiz sobre Cibersegurança

1. O que é um ataque DDoS?

  • Um ataque que sobrecarrega o servidor com tráfego de dados
  • Uma tentativa de obter acesso não autorizado a um sistema
  • Uma técnica de criptografia de dados

2. O que é phishing?

  • Um tipo de fraude onde se tenta obter informações sensíveis
  • Um ataque que sequestra o sistema operacional
  • Uma técnica para limpar dados do sistema

3. O que significa a sigla VPN?

  • Virtual Private Network (Rede Privada Virtual)
  • Very Private Network (Rede Muito Privada)
  • Virtual Protection Node (Nó de Proteção Virtual)

4. O que é um firewall?

  • Uma barreira de segurança que monitora o tráfego de dados
  • Uma ferramenta para criptografar dados em sistemas operacionais
  • Um tipo de malware que invade sistemas de redes

5. O que é um malware?

  • Um software malicioso projetado para danificar ou roubar dados
  • Um sistema de segurança usado para proteger redes corporativas
  • Uma ferramenta para melhorar a velocidade de conexão de redes

6. O que é criptografia?

  • Um processo de converter dados em um formato ilegível para proteger informações
  • Uma técnica de compressão de dados
  • Um método de aumentar a velocidade de um sistema de rede

7. O que é autenticação de dois fatores (2FA)?

  • Uma camada extra de segurança onde o usuário precisa de algo mais além da senha
  • Um método de recuperação de senha através de e-mail
  • Uma técnica para reduzir o tráfego de dados da rede

8. O que é um ataque de engenharia social?

  • Uma tentativa de manipular pessoas para obter informações confidenciais
  • Um ataque físico à infraestrutura de TI
  • Um tipo de ataque usando a criptografia para esconder dados

9. O que é um ataque Man-in-the-Middle (MitM)?

  • Um ataque onde o atacante intercepta a comunicação entre duas partes
  • Um ataque que altera o conteúdo de dados em trânsito
  • Um ataque que desativa o sistema de defesa de redes

10. O que é uma senha forte?

  • Uma senha com uma combinação de letras, números e caracteres especiais
  • Uma senha composta apenas por números sequenciais
  • Uma senha de apenas uma palavra comum
