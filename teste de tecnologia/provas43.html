Quiz sobre Data Engineering

1. O que é Data Engineering?

  • É o processo de construir e manter sistemas que coletam, armazenam e analisam grandes volumes de dados.
  • É o processo de treinar modelos de aprendizado de máquina para fazer previsões sobre grandes volumes de dados.
  • É o estudo de como melhorar a visualização de dados em tempo real.

2. O que é um pipeline de dados?

  • É um conjunto de processos automáticos para coletar, transformar e armazenar dados de forma eficiente.
  • É um processo de visualização de dados onde os dados são organizados em gráficos interativos.
  • É uma técnica de modelagem de dados usada em sistemas de recomendação.

3. O que é ETL no contexto de Data Engineering?

  • ETL significa Extrair, Transformar e Carregar, um processo que coleta dados de várias fontes, os transforma e os armazena em um data warehouse.
  • ETL é um tipo de algoritmo usado para otimizar consultas em bancos de dados.
  • ETL é uma técnica usada para prever tendências a partir de grandes volumes de dados.

4. Qual é a principal diferença entre Data Engineering e Data Science?

  • Data Engineering foca em preparar os dados para análise, enquanto Data Science foca em analisar os dados para extrair insights.
  • Data Engineering trabalha com modelos preditivos, enquanto Data Science foca apenas na visualização de dados.
  • Data Engineering é responsável apenas pela criação de interfaces de usuário para dados, enquanto Data Science foca na coleta dos dados.

5. O que é um Data Warehouse?

  • É um sistema utilizado para armazenar grandes volumes de dados de forma organizada e estruturada para facilitar a análise.
  • É um banco de dados especializado em armazenar dados temporários e em tempo real.
  • É um sistema que processa dados em tempo real para fornecer informações analíticas de forma instantânea.

6. O que é Hadoop?

  • Hadoop é um framework de software que permite o processamento e o armazenamento de grandes volumes de dados de forma distribuída.
  • Hadoop é uma linguagem de programação usada para criar pipelines de dados rápidos.
  • Hadoop é um sistema de banco de dados para armazenar dados temporários durante a execução de processos de Machine Learning.

7. O que é NoSQL?

  • NoSQL é um tipo de banco de dados que não utiliza o modelo relacional e é otimizado para dados não estruturados e escalabilidade horizontal.
  • NoSQL é uma linguagem de consulta utilizada em bancos de dados relacionais para otimizar o desempenho das consultas.
  • NoSQL é uma técnica de análise de dados em tempo real.

8. O que é Spark?

  • Spark é um framework de processamento de dados distribuído, rápido e de código aberto, usado para análises em larga escala.
  • Spark é um sistema de banco de dados relacional usado em sistemas de recomendação.
  • Spark é uma ferramenta de visualização de dados usada em Data Science.

9. O que é Data Lake?

  • Data Lake é um repositório de dados que armazena grandes volumes de dados em seu formato bruto e sem estrutura.
  • Data Lake é um sistema de processamento de dados em tempo real com alta disponibilidade.
  • Data Lake é um modelo de banco de dados relacional usado para armazenar dados temporários.

10. O que é um Data Pipeline?

  • É uma série de processos automatizados que capturam, processam e armazenam dados em diferentes sistemas de maneira eficiente.
  • É uma arquitetura de banco de dados projetada para melhorar o desempenho de consultas complexas.
  • É uma ferramenta para visualização de grandes volumes de dados em tempo real.
