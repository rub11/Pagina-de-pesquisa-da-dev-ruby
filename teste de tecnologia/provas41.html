Quiz sobre Python Avançado

1. O que é um *decorator* em Python?

  • Uma função que permite modificar o comportamento de outra função.
  • Uma estrutura de dados utilizada para armazenar valores temporários.
  • Uma técnica de otimização de código que reduz o tempo de execução.

2. O que significa a expressão *lambda* em Python?

  • Uma maneira de definir funções anônimas de forma simples.
  • Uma classe abstrata para representar operações matemáticas.
  • Uma função que permite realizar chamadas recursivas.

3. O que é um *gerador* em Python?

  • Uma função que utiliza a palavra-chave *yield* para retornar valores de forma preguiçosa.
  • Uma função que gera números aleatórios durante a execução.
  • Uma estrutura de dados que armazena valores em uma sequência infinita.

4. O que é *multiprocessing* em Python?

  • Uma biblioteca que permite a execução de múltiplos processos simultaneamente.
  • Um módulo que facilita a execução de tarefas em paralelo com threads.
  • Uma técnica de otimização para melhorar a performance de loops em grandes volumes de dados.

5. O que é o conceito de *metaclasses* em Python?

  • Uma classe que define o comportamento de outras classes.
  • Um tipo de função utilizada para gerenciar a execução de métodos em uma classe.
  • Uma estrutura de dados utilizada para a criação de instâncias de objetos de forma dinâmica.

6. O que é *list comprehension* em Python?

  • Uma maneira compacta de criar listas com base em uma expressão.
  • Uma forma de iterar sobre listas de maneira eficiente.
  • Uma técnica utilizada para criar objetos imutáveis em listas.

7. O que são *funções recursivas* em Python?

  • Funções que chamam a si mesmas durante sua execução.
  • Funções que retornam valores complexos baseados em operações matemáticas.
  • Funções que utilizam parâmetros em múltiplas formas para manipular dados.

8. O que é *duck typing* em Python?

  • Um estilo de tipagem dinâmica onde o tipo de um objeto é determinado por seu comportamento.
  • Uma técnica de otimização de código para melhorar a leitura de arquivos grandes.
  • Uma forma de tipagem que permite misturar diferentes tipos de dados em uma lista.

9. O que é *decorator pattern* em Python?

  • Uma técnica de design para adicionar funcionalidades a objetos de maneira flexível.
  • Uma estrutura que permite agrupar diferentes tipos de funções em um módulo.
  • Uma maneira de definir funções com parâmetros variáveis.

10. O que é *asyncio* em Python?

  • Uma biblioteca que permite a execução assíncrona de código para tarefas I/O-bound.
  • Uma estrutura para facilitar a execução paralela de código em múltiplos processos.
  • Uma técnica para otimizar a execução de laços em grandes volumes de dados.
