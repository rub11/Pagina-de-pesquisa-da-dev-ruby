Quiz sobre Data Science

O que é Data Science?

  • O campo que envolve a extração de conhecimento e insights de grandes volumes de dados.
  • Uma técnica que analisa dados em tempo real sem armazená-los.
  • Uma ferramenta usada exclusivamente para visualização de dados.

Quais são as etapas principais do processo de Data Science?

  • Coleta de dados, limpeza, análise, visualização e modelagem preditiva.
  • Coleta de dados, visualização e arquivamento de dados.
  • Somente a visualização de dados.

Qual linguagem de programação é amplamente usada em Data Science?

  • Python.
  • JavaScript.
  • PHP.

O que é uma variável dependente em um modelo de Data Science?

  • É a variável que se tenta prever, com base em outras variáveis independentes.
  • São variáveis que não têm relação com a variável dependente.
  • São variáveis que não podem ser alteradas ou manipuladas durante a análise.

Qual é a diferença entre aprendizado supervisionado e não supervisionado?

  • No aprendizado supervisionado, o modelo é treinado com dados rotulados, enquanto no não supervisionado, os dados não são rotulados.
  • No aprendizado supervisionado, o modelo não recebe nenhum dado de entrada.
  • No aprendizado não supervisionado, o modelo utiliza apenas aprendizado por reforço.

O que é uma variável independente em um modelo de Data Science?

  • São as variáveis usadas para prever a variável dependente.
  • São variáveis que não influenciam o modelo.
  • São variáveis que têm um impacto negativo no modelo.

O que significa overfitting em um modelo de Data Science?

  • Overfitting ocorre quando um modelo aprende demais com os dados de treinamento, tornando-se menos eficiente com novos dados.
  • Overfitting ocorre quando o modelo não é capaz de aprender nada dos dados de treinamento.
  • Overfitting ocorre quando um modelo é simples demais para fazer previsões precisas.

O que são algoritmos de classificação em Data Science?

  • Algoritmos que atribuem rótulos ou categorias a dados com base em características ou padrões.
  • Algoritmos usados apenas para visualizar dados.
  • Algoritmos usados para reduzir o número de variáveis no modelo.

O que é a técnica de regressão em Data Science?

  • Uma técnica usada para modelar e analisar a relação entre variáveis independentes e uma variável dependente contínua.
  • Uma técnica que simplesmente organiza os dados em grupos.
  • Uma técnica que aplica algoritmos de classificação a dados numéricos.

Qual é a importância da visualização de dados em Data Science?

  • A visualização de dados ajuda a entender padrões, detectar anomalias e comunicar insights de maneira mais eficiente.
  • A visualização de dados não é relevante em Data Science.
  • A visualização de dados é útil apenas para agradar visualmente, sem agregar informações significativas.
