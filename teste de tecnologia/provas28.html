Quiz sobre Programação Orientada a Objetos

1. O que é um objeto em POO?

  • Uma instância de uma classe com estado e comportamento.
  • Um arquivo de texto com funções dentro.
  • Um tipo especial de função recursiva.

2. Qual é o propósito de uma classe?

  • Servir como molde para criar objetos.
  • Organizar arquivos do projeto.
  • Guardar valores de entrada do usuário.

3. Qual é o princípio que permite esconder detalhes internos de implementação?

  • Encapsulamento
  • Herança
  • Polimorfismo

4. O que é herança em POO?

  • Mecanismo pelo qual uma classe pode herdar atributos e métodos de outra.
  • A forma de duplicar objetos.
  • O processo de copiar funções entre projetos.

5. O que é polimorfismo?

  • Capacidade de um objeto se comportar de várias formas com base no contexto.
  • Capacidade de copiar objetos.
  • Criação de múltiplas classes simultaneamente.

6. Qual dos itens abaixo é um exemplo de encapsulamento?

  • Atributos privados com métodos públicos de acesso.
  • Herança múltipla em classes.
  • Sobrescrita de métodos herdados.

7. O que define a visibilidade de um atributo ou método?

  • Modificadores de acesso como public, private, protected.
  • O tipo da variável (int, string etc).
  • A linguagem de programação utilizada.

8. O que é um construtor em uma classe?

  • Um método especial chamado ao instanciar a classe.
  • Um atributo usado para armazenar dados.
  • Uma exceção lançada ao criar objetos.

9. O que é a sobrescrita de métodos?

  • Modificar o comportamento de um método herdado em uma subclasse.
  • Criar múltiplos métodos com o mesmo nome.
  • Excluir um método da superclasse.

10. O que é uma interface?

  • Um contrato que define métodos que uma classe deve implementar.
  • Uma função anônima de inicialização de objeto.
  • Um tipo especial de herança múltipla.
