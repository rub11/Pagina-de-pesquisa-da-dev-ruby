Quiz sobre Redes Neurais Convolucionais

O que são redes neurais convolucionais?

  • São redes neurais especializadas no processamento de dados em forma de grade, como imagens.
  • São redes que apenas processam dados sequenciais, como textos e áudio.
  • São redes que processam apenas dados numéricos.

Qual é a principal característica das CNNs?

  • O uso de filtros convolucionais para extrair características de imagens ou sinais.
  • A capacidade de processar apenas informações textuais.
  • O processamento sequencial de dados em uma ordem específica.

Quais camadas compõem uma rede neural convolucional?

  • Camadas convolucionais, camadas de pooling e camadas totalmente conectadas.
  • Somente camadas convolucionais e de normalização.
  • Somente camadas de pooling e camadas ocultas.

O que é uma camada de pooling?

  • Uma camada que reduz a resolução das imagens, mantendo as informações mais importantes.
  • Uma camada que aumenta a resolução das imagens para melhorar a qualidade.
  • Uma camada que aplica filtros de forma similar às camadas convolucionais.

Quais são os principais tipos de pooling usados em CNNs?

  • Max pooling e average pooling.
  • Mean pooling e deep pooling.
  • Random pooling e deep convolutional pooling.

O que é uma operação de convolução?

  • Uma operação matemática que aplica um filtro sobre uma entrada, gerando um mapa de características.
  • Uma operação que apenas soma os valores de uma entrada para gerar a saída.
  • Uma operação de multiplicação entre todos os valores de uma imagem.

Qual é a função de ativação mais comum em CNNs?

  • ReLU (Rectified Linear Unit).
  • Sigmoid.
  • TanH.

Como as redes neurais convolucionais são aplicadas em visão computacional?

  • Elas são usadas para reconhecer objetos e características em imagens e vídeos.
  • Elas são usadas apenas para interpretar sequências de áudio.
  • Elas são usadas para transformar imagens em gráficos de dados.

Quais são os principais desafios ao treinar uma CNN?

  • Overfitting, falta de dados e alto custo computacional.
  • Baixo custo computacional e falta de precisão.
  • Baixa velocidade de processamento e necessidade de grandes bancos de dados de treinamento.

Quais são as vantagens de usar CNNs para tarefas de reconhecimento de imagem?

  • Elas são capazes de aprender automaticamente as características importantes das imagens sem necessidade de extração manual.
  • Elas precisam de menos dados para treinamento do que outros modelos.
  • Elas são mais rápidas que outros modelos, como redes neurais artificiais convencionais.
