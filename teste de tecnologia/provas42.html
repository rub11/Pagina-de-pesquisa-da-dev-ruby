Quiz sobre Redes Neurais Artificiais

1. O que é uma rede neural artificial?

  • Um sistema computacional inspirado no funcionamento do cérebro humano, projetado para reconhecer padrões.
  • Uma rede de computadores interconectados que processam dados em tempo real.
  • Um modelo de aprendizado de máquina baseado em árvores de decisão.

2. O que é uma camada de entrada em uma rede neural?

  • A camada que recebe os dados de entrada e os passa para as camadas seguintes.
  • A camada responsável por fornecer a saída final da rede neural.
  • A camada que processa os dados de saída e retorna ao usuário.

3. O que são pesos em uma rede neural?

  • Os valores que determinam a importância de cada entrada na rede neural.
  • As variáveis de controle que ajustam a saída da rede.
  • A quantidade de neurônios em cada camada da rede.

4. O que é a função de ativação em uma rede neural?

  • A função que determina se um neurônio será ativado ou não, baseando-se na entrada.
  • A função responsável por determinar a arquitetura da rede neural.
  • A função que calcula os valores de entrada para a rede neural.

5. O que é o algoritmo de retropropagação (backpropagation)?

  • O algoritmo usado para ajustar os pesos da rede neural durante o treinamento.
  • O algoritmo responsável por melhorar a arquitetura da rede.
  • O algoritmo que define a função de ativação em uma rede neural.

6. O que é uma rede neural convolucional (CNN)?

  • Um tipo de rede neural projetada para processar dados com uma grade regular, como imagens.
  • Uma rede neural com múltiplas camadas de entrada e saída.
  • Uma rede neural especializada em tarefas de processamento de texto.

7. O que é overfitting em uma rede neural?

  • Quando o modelo aprende detalhes excessivos dos dados de treinamento e perde a capacidade de generalizar.
  • Quando o modelo falha em aprender padrões dos dados de treinamento e tem desempenho ruim.
  • Quando o modelo aprende de forma eficiente sem perda de dados durante o treinamento.

8. O que são redes neurais recorrentes (RNN)?

  • Redes neurais que possuem conexões direcionadas para trás, permitindo que a rede tenha memória de eventos anteriores.
  • Redes neurais compostas por camadas separadas que processam dados em paralelo.
  • Redes neurais que se concentram exclusivamente em análise de grandes volumes de dados.

9. O que é uma rede neural densa?

  • Uma rede neural onde cada neurônio de uma camada está conectado a todos os neurônios da camada seguinte.
  • Uma rede neural onde os neurônios são conectados apenas de maneira hierárquica.
  • Uma rede neural que utiliza dados de entrada simples para o treinamento.

10. O que é *dropout* em uma rede neural?

  • Uma técnica para evitar overfitting, onde algumas conexões são desativadas aleatoriamente durante o treinamento.
  • Uma técnica para aumentar a capacidade computacional durante o treinamento.
  • Uma técnica de regularização que aumenta a complexidade da rede neural.
