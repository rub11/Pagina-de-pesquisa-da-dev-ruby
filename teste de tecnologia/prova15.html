Quiz sobre Big Data

1. O que significa Big Data?

Conjunto de dados pequenos
Conjunto de dados em grande volume, variedade e velocidade
Conjunto de dados limitados

2. Qual é a principal característica do Big Data?

Capacidade de armazenar dados apenas locais
Capacidade de processar grandes volumes de dados em tempo real
Capacidade de processar pequenas quantidades de dados de forma rápida

3. O que é um exemplo de ferramenta usada para Big Data?

Excel
Hadoop
Notepad

4. Qual é o conceito de "velocidade" em Big Data?

Rapidez na análise de dados
Rapidez na coleta de dados em tempo real
Rapidez na realização de tarefas computacionais

5. O que são "dados não estruturados"?

Dados organizados em formato de tabelas
Dados que não seguem um formato predefinido, como textos e imagens
Dados coletados apenas de fontes estruturadas como bancos de dados

6. Quais empresas são conhecidas por usarem Big Data?

Google, Facebook e Amazon
Microsoft, Oracle e SAP
Apple, Spotify e Twitter

7. O que é "variedade" em Big Data?

Diversidade de tipos de dados coletados
Tamanho dos dados processados
Rapidez na obtenção de dados

8. O que é uma "base de dados NoSQL"?

Uma base de dados relacional
Uma base de dados não relacional, ideal para Big Data
Uma base de dados usada apenas para dados financeiros

9. O que significa "análise preditiva" em Big Data?

Análise de dados em tempo real para prever o futuro
Análise de dados históricos para prever tendências futuras
Análise de dados em tempo real sem previsões

10. Qual é a importância de Big Data para as empresas?

Permite reduzir custos operacionais
Ajuda na tomada de decisões estratégicas baseadas em dados
Aumenta a complexidade das operações diárias
