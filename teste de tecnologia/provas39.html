Quiz sobre Programação PHP

1. O que significa PHP?

  • PHP: Hypertext Preprocessor
  • Personal Home Page
  • PHP: Programmatic Hypertext

2. Como você escreve um comentário de uma linha em PHP?

  • // Este é um comentário
  • # Este é um comentário
  • /* Este é um comentário */

3. Qual função é usada para incluir arquivos em PHP?

  • include()
  • require()
  • import()

4. O que a função `echo` faz em PHP?

  • Imprime uma ou mais strings
  • Realiza operações matemáticas
  • Declara uma variável

5. O que é um array em PHP?

  • Uma variável que pode armazenar múltiplos valores
  • Uma função matemática em PHP
  • Um tipo de dado que só armazena números

6. Como você define uma variável em PHP?

  • $nome = "Valor"
  • var nome = "Valor"
  • def nome = "Valor"

7. Como você pode criar uma função em PHP?

  • function nomeFunção() {}
  • função nomeFunção() {}
  • def nomeFunção() {}

8. O que significa a palavra-chave `public` em PHP?

  • Acessível de qualquer lugar
  • Acessível apenas dentro da classe
  • Acessível apenas dentro do mesmo arquivo

9. Como você define uma constante em PHP?

  • define('NOME_CONSTANTE', 'valor');
  • const NOME_CONSTANTE = 'valor';
  • const('NOME_CONSTANTE', 'valor');

10. Qual a principal diferença entre `include` e `require`?

  • `include` gera um aviso, enquanto `require` gera um erro fatal
  • Não há diferença
  • `include` não pode ser utilizado em arquivos externos
