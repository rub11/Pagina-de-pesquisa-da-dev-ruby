Quiz sobre Testes de Software

1. Qual é o principal objetivo dos testes de software?

  • Identificar defeitos e garantir a qualidade do software.
  • Aumentar o número de linhas de código.
  • Melhorar o desempenho do hardware.

2. O que é teste unitário?

  • Teste que verifica o comportamento de unidades isoladas de código.
  • Teste de usabilidade com usuários.
  • Teste feito ao final do projeto.

3. O que é um bug?

  • Um erro ou defeito no software.
  • Um estilo de programação.
  • Um tipo de teste automatizado.

4. O que caracteriza um teste de integração?

  • Verifica a interação entre diferentes módulos do sistema.
  • Analisa o design de interface do usuário.
  • Testa conexões físicas entre servidores.

5. O que são testes funcionais?

  • Testes baseados nos requisitos do sistema.
  • Testes da performance do hardware.
  • Testes feitos apenas no frontend.

6. O que são testes automatizados?

  • Testes executados automaticamente por ferramentas ou scripts.
  • Testes feitos por humanos sem uso de ferramentas.
  • Simulações manuais em tempo real.

7. O que é TDD (Test Driven Development)?

  • Desenvolvimento orientado por testes, onde os testes são escritos antes do código.
  • Teste de código já finalizado.
  • Uso de banco de dados de teste.

8. O que são testes de regressão?

  • Testes para verificar se novas mudanças não quebraram funcionalidades existentes.
  • Testes feitos com dados fictícios.
  • Testes apenas em versões antigas do software.

9. Quais ferramentas são comuns para testes automatizados?

  • JUnit, Selenium, Cypress.
  • Photoshop, Illustrator, Canva.
  • Excel, Word, PowerPoint.

10. Qual o benefício principal da automação de testes?

  • Maior rapidez e confiabilidade na execução de testes repetitivos.
  • Criação de código mais complexo.
  • Geração automática de documentação.
