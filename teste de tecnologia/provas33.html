Quiz sobre Programação C

1. Qual é a extensão padrão de arquivos C?

2. Qual função é o ponto de entrada de um programa em C?

3. Como se declara uma variável inteira?

5. Como imprimir algo na tela em C?

  • print()

6. Qual operador é usado para comparar igualdade?

7. Qual palavra-chave é usada para criar loops em C?

10. Qual símbolo é usado para apontadores em C?
