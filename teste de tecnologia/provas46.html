Quiz sobre Computação Paralela

1. O que é Computação Paralela?

  • É a técnica de usar múltiplos processadores para resolver um problema simultaneamente, dividindo as tarefas.
  • É a execução de programas sequenciais, um após o outro, sem dividir a carga de trabalho.
  • É a técnica de aumentar a velocidade do processador para melhorar a performance do programa.

2. Qual é a principal vantagem da Computação Paralela?

  • Permite a execução mais rápida de tarefas, dividindo a carga de trabalho entre vários processadores.
  • Aumenta a complexidade do programa, tornando-o mais difícil de implementar.
  • Reduz o custo de execução de programas em servidores.

3. O que são threads em Computação Paralela?

  • São unidades de execução que podem ser executadas simultaneamente, dentro de um mesmo processo.
  • São recursos físicos que são alocados para aumentar a capacidade de memória.
  • São módulos de armazenamento de dados que ajudam a acelerar a computação.

4. O que é a Lei de Amdahl em Computação Paralela?

  • É uma fórmula que descreve a melhoria teórica da performance de um sistema paralelo com a adição de processadores.
  • É uma técnica que permite a multiplicação de tarefas em máquinas com um único processador.
  • É uma lei que trata da limitação de largura de banda em sistemas de computação distribuída.

5. O que é um modelo de memória compartilhada?

  • É um modelo em que múltiplos processadores acessam a mesma memória principal.
  • É um modelo em que cada processador tem sua própria memória, sem comunicação entre eles.
  • É um modelo onde a memória é alocada de forma dinâmica entre os processadores conforme a demanda.

6. O que é o modelo de passagem de mensagens?

  • É um modelo onde os processadores se comunicam enviando e recebendo mensagens entre si.
  • É um modelo onde os processadores têm memória compartilhada e não precisam trocar mensagens.
  • É um modelo de execução sequencial onde não há comunicação entre os processadores.

7. O que é um supercomputador?

  • É um computador projetado para realizar cálculos extremamente rápidos e complexos, utilizando Computação Paralela.
  • É um computador com grande capacidade de armazenamento, mas sem suporte para Computação Paralela.
  • É um computador comum, mas com capacidade de suportar múltiplos usuários simultaneamente.

8. Qual é a diferença entre Computação Paralela e Computação Distribuída?

  • Computação Paralela envolve a execução simultânea de processos no mesmo computador, enquanto Computação Distribuída envolve a execução de processos em vários computadores.
  • Computação Paralela é mais eficaz, pois envolve apenas computadores centralizados.
  • Não há diferença significativa entre Computação Paralela e Computação Distribuída.

9. O que é uma condição de corrida em Computação Paralela?

  • É um erro que ocorre quando múltiplos processos tentam acessar ou modificar dados simultaneamente, resultando em um comportamento inesperado.
  • É uma condição em que todos os processos devem esperar um pelo outro para garantir a execução sequencial.
  • É uma situação em que o sistema opera mais rapidamente, sem qualquer erro ou concorrência.

10. O que é o modelo de programação CUDA?

  • É uma plataforma e modelo de programação que permite que os programadores usem GPUs para executar cálculos paralelos.
  • É uma técnica de programação para otimizar o desempenho de CPUs em ambientes de computação sequenciais.
  • É um modelo de programação voltado para aumentar a capacidade de armazenamento de dados em computadores.
