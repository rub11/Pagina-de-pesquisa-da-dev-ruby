Quiz sobre Design Responsivo

1. O que é design responsivo?

  • Um design que se adapta a diferentes dispositivos
  • Um design que responde ao servidor
  • Um design exclusivo para desktop

2. Qual tecnologia é essencial para design responsivo?

  • Media Queries
  • SQL
  • AJAX

3. O que define um layout fluido?

  • Uso de porcentagens em vez de pixels fixos
  • Designs que usam cores vibrantes
  • Elementos que se movem aleatoriamente

4. O que são breakpoints em design responsivo?

  • Pontos onde o layout muda para se adaptar à tela
  • Erros no carregamento da página
  • Interrupções na rede

5. Qual dessas práticas ajuda no design responsivo?

  • Utilizar unidades relativas como em (%) ou vw/vh
  • Trabalhar apenas com imagens em alta resolução
  • Fixar o tamanho dos elementos com px

6. Qual o benefício de um site responsivo?

  • Melhor experiência do usuário em qualquer dispositivo
  • Mais uso de memória
  • Aumento do número de scripts

7. Qual framework facilita o design responsivo?

  • Bootstrap
  • jQuery
  • Flask

8. O que são imagens responsivas?

  • Imagens que se adaptam ao tamanho da tela
  • Imagens com animações
  • Imagens em preto e branco

9. Qual unidade é mais usada para fontes em design responsivo?

  • em ou rem
  • px
  • cm

10. Como garantir que o site fique responsivo em dispositivos móveis?

  • Usando viewport meta tag
  • Removendo todos os scripts
  • Utilizando apenas tabelas
