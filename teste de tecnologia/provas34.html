Quiz sobre Design de Interface de Usuário

1. O que é UI no contexto de design digital?

  • User Interface
  • User Integration
  • Universal Interaction

2. Qual é o principal objetivo do design de interface de usuário?

  • Facilitar a interação com o sistema
  • Tornar o código mais eficiente
  • Reduzir o tamanho do software

3. Um bom design de UI deve ser:

  • Intuitivo e consistente
  • Complexo e sofisticado
  • Colorido e chamativo

4. Qual ferramenta é amplamente usada no design de UI?

  • Figma
  • Excel
  • Notepad

5. O que significa o termo "wireframe" no design de UI?

  • Esboço da estrutura da interface
  • Código da interface
  • Teste de usabilidade

6. Qual desses princípios é essencial no design de UI?

  • Consistência
  • Redundância
  • Aleatoriedade

7. O que são protótipos no design de UI?

  • Simulações navegáveis da interface
  • Códigos finais de produção
  • Relatórios de bugs

8. O que o termo "responsivo" significa em UI?

  • Adapta-se a diferentes dispositivos
  • Responde apenas em telas grandes
  • Tem muitas animações

9. Qual é a vantagem de seguir padrões de design?

  • Melhora a usabilidade e reduz erros
  • Torna o design mais original
  • Facilita o uso de efeitos visuais

10. O que significa “feedback visual” em uma interface?

  • Reação visual após uma ação do usuário
  • Resposta do servidor
  • Notificações por e-mail
