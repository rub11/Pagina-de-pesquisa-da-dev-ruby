Quiz sobre Game Engines

1. O que é uma Game Engine?

  • Um software que fornece ferramentas para desenvolvimento de jogos.
  • Um console de videogame portátil.
  • Um tipo de joystick avançado.

2. Qual é uma das Game Engines mais populares para desenvolvimento 2D e 3D?

  • Unity
  • Photoshop
  • PowerPoint

3. O que a Unreal Engine é conhecida por oferecer?

  • Gráficos de alta qualidade e motor poderoso.
  • Edição de planilhas avançada.
  • Renderização de áudio apenas.

4. Qual linguagem de programação é usada comumente na Unity?

  • C#
  • Java
  • HTML

5. Qual engine é frequentemente usada em jogos mobile e indie?

  • Godot
  • Autodesk Maya
  • Notepad++

6. O que é o motor de física em uma engine?

  • Componente que simula gravidade, colisões e forças.
  • Um acelerador de áudio.
  • Ferramenta de design gráfico.

7. Qual engine é mais utilizada para jogos AAA como Fortnite?

  • Unreal Engine
  • Pygame
  • Scratch

8. Qual vantagem principal das game engines?

  • Facilitam o desenvolvimento ao fornecer ferramentas prontas.
  • Imprimem jogos automaticamente.
  • Eliminam a necessidade de programação completamente.

9. Qual dessas engines é open-source?

  • Godot
  • Unity
  • CryEngine

10. Qual recurso é comum em engines modernas?

  • Renderização em tempo real.
  • Geração de relatórios fiscais.
  • Conversão automática para PDF.
