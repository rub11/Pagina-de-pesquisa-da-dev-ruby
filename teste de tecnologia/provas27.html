Quiz sobre Containers e Kubernetes

1. O que é um container?

  • Uma unidade leve e portátil que embala software e suas dependências.
  • Um tipo especial de máquina virtual com sistema operacional completo.
  • Uma partição de disco rígido usada para isolar arquivos do sistema.

2. Qual é a principal ferramenta para criar containers?

  • Docker
  • Kubernetes
  • Nginx

3. O que é o Kubernetes?

  • Uma plataforma de orquestração de containers para gerenciar e escalar aplicações.
  • Um sistema operacional voltado para ambientes cloud.
  • Uma ferramenta para compilar imagens de containers.

4. Qual recurso do Kubernetes mantém a aplicação sempre em execução?

  • ReplicaSet
  • Service
  • ConfigMap

5. O que é um Pod no Kubernetes?

  • A menor unidade executável no Kubernetes, que pode conter um ou mais containers.
  • Uma máquina virtual leve criada dentro do cluster Kubernetes.
  • Uma imagem Docker pronta para ser executada.

6. Qual comando do Docker cria uma imagem?

  • docker build
  • docker run
  • docker pull

7. O que é um Deployment no Kubernetes?

  • Um recurso que define como uma aplicação deve ser implantada e atualizada.
  • Um container isolado em execução.
  • Um banco de dados de configurações externas.

8. Qual é a função de um Service no Kubernetes?

  • Expõe os Pods como um serviço acessível dentro ou fora do cluster.
  • Cria volumes persistentes para os Pods.
  • Armazena logs de execução dos Pods.

9. O que é um volume em Kubernetes?

  • Um espaço de armazenamento persistente acessado pelos containers.
  • Um tipo especial de rede entre containers.
  • Uma camada de abstração entre o cluster e o usuário.

10. Qual vantagem os containers oferecem sobre as máquinas virtuais?

  • Maior leveza, rapidez no boot e menor consumo de recursos.
  • Possibilidade de usar sistemas operacionais diferentes em um mesmo host.
  • Maior isolamento de hardware físico.
