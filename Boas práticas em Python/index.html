Boas Práticas em Python

Escrevendo código limpo, legível e pythonicamente elegante

1. Siga a PEP 8

A PEP 8 é o guia oficial de estilo para Python. Ela cobre espaçamento, nomes, indentação e organização.

# Correto
def calcular_total(valor, taxa):
    return valor + (valor * taxa)

2. Use nomes claros e significativos

Evite abreviações desnecessárias. Dê nomes que explicam a função da variável ou método.

# Ruim
def ct(v, t):
    return v + (v * t)

# Bom
def calcular_total(valor, taxa):
    return valor + (valor * taxa)

3. Comente com moderação

Comente o porquê do código, não o que ele faz. Códigos bem escritos precisam de poucos comentários.

# Aplica 10% de desconto no preço do produto
def aplicar_desconto(preco):
    return preco * 0.9

4. Escreva funções pequenas e específicas

Funções devem realizar apenas uma tarefa. Isso torna o código mais fácil de testar e manter.

5. Utilize listas por compreensão

Sintaxe concisa e legível para criar listas a partir de iteráveis.

# Tradicional
quadrados = []
for i in range(10):
    quadrados.append(i * i)

# Compreensão
quadrados = [i * i for i in range(10)]

6. Trate exceções corretamente

Evite except Exception genérico. Capture exceções específicas e forneça mensagens úteis.

try:
    resultado = 10 / divisor
except ZeroDivisionError:
    print("Erro: divisão por zero.")

7. Utilize f-strings

São mais modernas, rápidas e legíveis para formatação de strings.

nome = "Ruby"
print(f"Bem-vinda, {nome}!")

8. Mantenha código DRY (Don't Repeat Yourself)

Evite duplicação criando funções ou abstrações reutilizáveis.

9. Tipagem opcional com type hints

Facilita a leitura e uso de ferramentas de análise estática.

def somar(a: int, b: int) -> int:
    return a + b

10. Ferramentas úteis

  • ✅ Black: formata seu código automaticamente
  • ✅ Pylint ou Flake8: análise de qualidade
  • ✅ MyPy: verificação de tipos
  • ✅ pytest: testes automatizados

Conclusão

Seguir boas práticas torna o código Python mais legível, confiável e profissional. O "Pythonic way" valoriza clareza, elegância e simplicidade.
