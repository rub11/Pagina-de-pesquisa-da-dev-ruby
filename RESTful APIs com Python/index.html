RESTful APIs com Python

Aprenda a criar e consumir APIs RESTful de forma eficiente usando Python. Vamos explorar as principais práticas, bibliotecas e exemplos práticos para você se tornar um especialista na construção de APIs.

O que são APIs RESTful?

APIs RESTful seguem os princípios do estilo arquitetônico REST (Representational State Transfer). Elas usam métodos HTTP como GET, POST, PUT e DELETE para manipulação de dados. REST é amplamente adotado por ser simples, eficiente e fácil de escalar.

Configurando o Ambiente

Antes de começarmos a construir uma API, é necessário configurar o ambiente. Usaremos o Flask, um micro-framework para Python, para criar nossa API de maneira rápida e fácil.

pip install flask

Criando a Primeira Rota

Agora, vamos criar nossa primeira rota API usando o Flask. Abaixo está o código básico para iniciar uma API RESTful simples.

from flask import Flask
app = Flask(__name__)

@app.route('/')
def home():
    return 'Olá, API RESTful com Python!'

if __name__ == '__main__':
    app.run(debug=True)

GET: Recuperando Dados

O método GET é utilizado para recuperar dados. A seguir, um exemplo de como criar uma rota GET para retornar informações em formato JSON.

@app.route('/dados', methods=['GET'])
def get_dados():
    return {'nome': 'João', 'idade': 30}

POST: Enviando Dados

O método POST é utilizado para enviar dados ao servidor. Vamos aprender a enviar dados via JSON usando o Flask.

from flask import request

@app.route('/adicionar', methods=['POST'])
def adicionar_dados():
    dados = request.get_json()
    return {'status': 'Dados recebidos', 'dados': dados}

PUT: Atualizando Dados

O método PUT é utilizado para atualizar dados existentes. Abaixo temos um exemplo de como criar uma rota PUT para atualizar informações.

@app.route('/atualizar', methods=['PUT'])
def atualizar_dados():
    dados = request.get_json()
    return {'status': 'Dados atualizados', 'dados': dados}

DELETE: Excluindo Dados

O método DELETE é utilizado para excluir dados. Vamos criar uma rota DELETE que remove um item do banco de dados ou do sistema.

@app.route('/excluir', methods=['DELETE'])
def excluir_dados():
    return {'status': 'Dados excluídos'}

Consumindo a API com Python

Agora que temos nossa API criada, podemos consumir os endpoints usando o módulo requests do Python. A seguir, um exemplo de como fazer uma requisição GET para recuperar dados.

import requests
response = requests.get('http://127.0.0.1:5000/dados')
print(response.json())

Conclusão

Com o Flask e a biblioteca requests, podemos criar e consumir APIs RESTful de maneira eficiente e simples. A partir deste ponto, você pode expandir suas APIs para incluir autenticação, conexão com bancos de dados e muito mais.
