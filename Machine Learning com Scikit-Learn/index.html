Machine Learning com Scikit-Learn

O Scikit-Learn é uma das bibliotecas mais populares para aprendizado de máquina em Python, oferecendo uma ampla gama de ferramentas para análise de dados, modelagem preditiva e avaliação de modelos.

O que é o Scikit-Learn?

Scikit-Learn é uma biblioteca de aprendizado de máquina que fornece ferramentas simples e eficientes para análise de dados, sendo ideal para iniciantes e especialistas. Ela oferece algoritmos para classificação, regressão, clustering, e muito mais.

Instalando o Scikit-Learn

Para começar a usar o Scikit-Learn, basta instalar a biblioteca com o pip:


pip install scikit-learn

Importando Bibliotecas

Para começar, você precisa importar o Scikit-Learn e outras bibliotecas úteis como o NumPy e o Pandas:


import numpy as np
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier

Exemplo de Classificação com RandomForest

Vamos treinar um modelo de classificação usando o algoritmo RandomForest:


# Carregar dados (exemplo)
data = pd.read_csv('dados.csv')

# Separar em variáveis independentes e dependentes
X = data.drop('target', axis=1)
y = data['target']

# Dividir em treino e teste
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# Criar e treinar o modelo
model = RandomForestClassifier()
model.fit(X_train, y_train)

# Prever e avaliar
predictions = model.predict(X_test)
print("Acurácia:", model.score(X_test, y_test))

Regressão com Scikit-Learn

O Scikit-Learn também permite realizar tarefas de regressão. Veja um exemplo usando o modelo de regressão linear:


from sklearn.linear_model import LinearRegression

# Criar o modelo de regressão
regressor = LinearRegression()

# Treinar o modelo
regressor.fit(X_train, y_train)

# Fazer previsões
y_pred = regressor.predict(X_test)
print("Acurácia:", regressor.score(X_test, y_test))

Clustering com K-Means

O K-Means é um algoritmo popular para clustering. Veja como aplicá-lo:


from sklearn.cluster import KMeans

# Criar o modelo de clustering
kmeans = KMeans(n_clusters=3)

# Ajustar o modelo aos dados
kmeans.fit(X)

# Ver os centros dos clusters
print(kmeans.cluster_centers_)

Avaliação de Modelos

A avaliação de modelos é essencial para medir a performance de seu modelo de aprendizado de máquina. O Scikit-Learn fornece diversas métricas como Acurácia, Precisão, Recall, etc.


from sklearn.metrics import accuracy_score, confusion_matrix

# Acurácia
accuracy = accuracy_score(y_test, predictions)
print("Acurácia:", accuracy)

# Matriz de Confusão
conf_matrix = confusion_matrix(y_test, predictions)
print("Matriz de Confusão:\n", conf_matrix)

Considerações Finais

Scikit-Learn oferece uma plataforma robusta para machine learning em Python. Com ela, você pode treinar modelos, avaliá-los e realizar predições de maneira simples e eficaz.
