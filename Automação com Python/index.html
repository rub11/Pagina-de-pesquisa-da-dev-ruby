Automação com Python

1. O que é Automação com Python?

Automação com Python é o processo de utilizar scripts Python para executar tarefas repetitivas, complexas ou demoradas de forma automática. Isso pode incluir atividades como mover arquivos, preencher planilhas, fazer scraping de dados de sites, interagir com APIs, automatizar tarefas no navegador, enviar e-mails automáticos e muito mais. Python é ideal para automação por sua sintaxe clara, grande biblioteca padrão e centenas de pacotes externos que facilitam essas integrações.

2. Exemplos Práticos de Automação com Python

2.1. Renomear Arquivos em Massa

Imagine que você tem centenas de arquivos em uma pasta e deseja renomeá-los automaticamente. 

import os

folder = "caminho/para/a/pasta"
for count, filename in enumerate(os.listdir(folder)):
    new_name = f"arquivo_{count}.txt"
    os.rename(os.path.join(folder, filename), os.path.join(folder, new_name))

2.2. Preencher uma Planilha Excel Automaticamente

Com a biblioteca openpyxl, é possível criar e manipular planilhas Excel. 

from openpyxl import Workbook

wb = Workbook()
ws = wb.active

# Inserindo dados
ws['A1'] = "Nome"
ws['B1'] = "Idade"
ws.append(["Alice", 30])
ws.append(["Bob", 25])

wb.save("dados.xlsx")

2.3. Automatizar o Envio de E-mails

Python permite enviar e-mails programaticamente com a biblioteca smtplib. 

import smtplib
from email.message import EmailMessage

msg = EmailMessage()
msg.set_content("Olá, isso é um e-mail enviado automaticamente.")
msg['Subject'] = "Automação com Python"
msg['From'] = "seuemail@exemplo.com"
msg['To'] = "destinatario@exemplo.com"

with smtplib.SMTP_SSL('smtp.gmail.com', 465) as smtp:
    smtp.login("seuemail@exemplo.com", "sua_senha")
    smtp.send_message(msg)

2.4. Web Scraping com BeautifulSoup

Capturar informações de um site para análise ou relatórios automatizados. 

import requests
from bs4 import BeautifulSoup

url = "https://quotes.toscrape.com"
res = requests.get(url)
soup = BeautifulSoup(res.text, 'html.parser')

for quote in soup.select(".quote"):
    texto = quote.select_one(".text").get_text()
    autor = quote.select_one(".author").get_text()
    print(f"{texto} - {autor}")
