Manipulação de Dados com Pandas

O Pandas é uma das bibliotecas mais populares para manipulação de dados em Python. Ele fornece estruturas de dados rápidas, flexíveis e expressivas, especialmente para trabalhar com tabelas e séries temporais.

O que é Pandas?

Pandas é uma biblioteca de código aberto que facilita a manipulação de dados, como leitura, limpeza e análise. As estruturas principais do Pandas são o DataFrame e a Series.

Instalando Pandas

Para começar a usar o Pandas, basta instalá-lo com o pip:


pip install pandas

Importando o Pandas

Após instalar, você pode importar o Pandas para seu código da seguinte maneira:


import pandas as pd

Criando um DataFrame

Um DataFrame é a estrutura de dados principal do Pandas. Ele pode ser criado a partir de um dicionário de listas ou arrays:


import pandas as pd

# Criando um DataFrame
dados = {'Nome': ['João', 'Maria', 'José'], 'Idade': [25, 30, 22]}
df = pd.DataFrame(dados)
print(df)

Visualizando Dados

Você pode visualizar as primeiras linhas de um DataFrame usando o método head():


print(df.head())

Filtrando Dados

Para filtrar dados, você pode usar condições dentro do DataFrame, como mostrado abaixo:


# Filtrando por idade
df_filtrado = df[df['Idade'] > 24]
print(df_filtrado)

Alterando Dados

Você também pode modificar os valores em um DataFrame. Aqui está um exemplo de como alterar a idade de uma pessoa:


# Alterando dados
df.loc[df['Nome'] == 'Maria', 'Idade'] = 35
print(df)

Operações Estatísticas

O Pandas facilita a realização de operações estatísticas, como a média, mediana, desvio padrão e muito mais:


# Cálculo da média
media_idade = df['Idade'].mean()
print('Média de idade:', media_idade)

Leitura e Escrita de Arquivos

Pandas facilita a leitura e escrita de dados em diversos formatos, como CSV, Excel, e SQL. Veja como carregar um arquivo CSV:


# Lendo um arquivo CSV
df_csv = pd.read_csv('arquivo.csv')
print(df_csv.head())

Considerações Finais

O Pandas é uma ferramenta poderosa para qualquer cientista de dados ou desenvolvedor que precise manipular grandes volumes de dados. Com ele, você pode fazer análise de dados rápida e eficientemente.
