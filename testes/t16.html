Quiz: Big Data

Big Data refere-se a conjuntos de dados muito grandes ou complexos que métodos tradicionais de processamento de dados não conseguem lidar de forma eficiente. Envolve características como grande volume, alta velocidade de geração e variedade de formatos.

Vamos testar seus conhecimentos sobre esse tema!

1. O que caracteriza o Big Data?

Grande volume de dados, processados rapidamente em tempo real
Somente dados armazenados em bancos de dados relacionais
Dados exclusivamente estruturados

2. Quais são as 3 principais características do Big Data?

Volume, Velocidade e Variedade
Segurança, Armazenamento e Processamento
Conectividade, Escalabilidade e Velocidade

3. Qual destas tecnologias é amplamente usada em Big Data?

Hadoop
Photoshop
Excel básico

4. O que significa "variedade" no contexto de Big Data?

Tipos diferentes de dados (texto, vídeo, imagem, etc.)
Diferentes formas de velocidade de transmissão
Variação dos preços de armazenamento

5. Qual das opções não é uma ferramenta usada em Big Data?

MySQL
Hadoop
Spark
