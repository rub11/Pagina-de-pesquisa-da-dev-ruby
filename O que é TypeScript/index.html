O que é TypeScript

O superconjunto de JavaScript que traz mais segurança e produtividade ao seu código

Introdução ao TypeScript

TypeScript é uma linguagem de programação desenvolvida pela Microsoft, que se baseia no JavaScript, mas adiciona tipagem estática e recursos avançados de desenvolvimento. É um superconjunto de JavaScript, ou seja, todo código JavaScript válido é também um código TypeScript válido.

Vantagens do TypeScript

  • Tipagem Estática: evita erros comuns ao validar os tipos das variáveis antes da execução.
  • Autocomplete e IntelliSense: editores modernos fornecem sugestões inteligentes com base nos tipos definidos.
  • Refatoração mais segura: mudar nomes de variáveis ou estruturas se torna mais seguro.
  • Orientação a Objetos: suporte a classes, interfaces e herança.
  • Melhor organização: permite uma arquitetura mais robusta em projetos grandes.

Exemplo comparativo

JavaScript:

function soma(a, b) {
  return a + b;
}

TypeScript:

function soma(a: number, b: number): number {
  return a + b;
}

No exemplo TypeScript, os tipos number são definidos, o que impede chamadas incorretas como soma("2", true).

Transpilação para JavaScript

TypeScript não é executado diretamente no navegador. Ele precisa ser transpilado para JavaScript através do comando:

tsc script.ts

Esse processo converte o código TypeScript em JavaScript padrão para execução em qualquer navegador.

Instalação

Você pode instalar o TypeScript globalmente com o seguinte comando via NPM:

npm install -g typescript

Conclusão

TypeScript é uma excelente escolha para projetos profissionais, especialmente em times e aplicações de grande escala. Ele traz confiabilidade, clareza e poder ao já versátil JavaScript.
